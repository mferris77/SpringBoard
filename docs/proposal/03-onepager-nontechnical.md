# KantarAssistant: AI for Your Team

## What is it?

An internal AI assistant that helps your team work smarter—not harder. Built on Microsoft's ecosystem, it leverages your existing Copilot Enterprise license and Azure infrastructure.

## Why Do We Need It?

Your team currently spends **2-4 hours per week** on:
- Manual Excel/Sheets updates
- Running the same reports over and over
- Answering the same questions
- Looking for scattered documentation

**KantarAssistant** automates all of this—while staying fully within Microsoft's security perimeter.

## What Can It Do?

| Task | Example | Time Saved |
|------|---------|-------------|
| 📊 **Reports** | "Generate weekly sales report" | 30 min/report |
| 📝 **Excel** | "Update this spreadsheet" | 20 min/task |
| 🔍 **Research** | "Find similar studies to X" | 1 hour/search |
| 💬 **Answers** | "How do I access the VPN?" | 10 min/query |
| 📣 **Teams** | "Notify team when data updates" | 15 min/alert |

## How Is It Different From Regular Copilot?

| Feature | KantarAssistant | Standard Copilot |
|---------|-----------------|------------------|
| **Custom Skills** | Builds skills for YOUR workflows | Generic only |
| **Internal Data** | Connects to your APIs, SQL | Limited M365 only |
| **Automation** | Triggers Power Automate flows | Read-only essentially |
| **Control** | Full ownership & customization | Microsoft-controlled |

## What Does It Cost?

- **Software**: Free (OpenClaw is open source)
- **Azure**: Uses existing Azure OpenAI credits (Kantar likely has via M365)
- **Infrastructure**: Minimal (can run in existing Azure VM)
- **Time to deploy**: 2-3 months
