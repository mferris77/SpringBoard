## Design Philosophy: Claude Code vs PI Agent

**Claude Code:**
- Great out-of-the-box defaults
- Low floor - anyone can jump in and start getting value
- Comes with safety modes
- 10,000 token system prompt with baked-in best practices
- More abstracted - you don't always see what's happening
- Prioritizes their cloud models
- For-profit company

**PI Agent:**
- Very minimal approach - "if I don't need it, it won't be built"
- No safety mode by default - runs in YOLO mode
- Only 200 token system prompt - "let the model cook"
- Wants to surface as much information as possible
- Doesn't care what model you use
- Open source - created by Mario Zechner
